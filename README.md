# notes-assignment

## Application setup
```
Clone the code repo
Navigate to project folder
Use node version 14.X 
do npm install
start the application using 'npm start'
Now application will be running on port 3000
```

## Test the appication using the postman APIs

```
Import API collection from notes-assignment/docs/postman into postman
Import environment from notes-assignment/docs/postman into postman

```

## Note
```
File size is limited to 10 MB, which can be modified in app.js
Application is developmed & tested in ubuntu 16.4
```
